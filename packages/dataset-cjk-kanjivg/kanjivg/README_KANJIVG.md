# KanjiVG SVG files

This directory contains SVG files sourced from the
[KanjiVG project](https://kanjivg.tagaini.net/),
[KanjiVG/kanjivg @ `r20250816`](https://github.com/KanjiVG/kanjivg/releases/tag/r20250816)
— Copyright (C) 2009–2025 Ulrich Apel and KanjiVG contributors — and licensed
under **Creative Commons Attribution-ShareAlike 3.0 Unported (CC-BY-SA 3.0)**.

Do **not** edit these files. They are regenerated from the pinned upstream
release by `bun run fetch-kanjivg`. Any local modifications will be discarded
the next time the script runs.

> ⚠️ **The SVGs themselves are not checked into git** — they are fetched on
> demand from the upstream release. See the top-level
> [`README.md`](../README.md) for the one-command refresh. The npm package
> tarball ships with the filtered SVG set included.
