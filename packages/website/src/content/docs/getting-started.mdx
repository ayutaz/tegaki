## Quick Start

<Steps>

1. **Pick a font bundle**

   Use one of the built-in font bundles:

   ```tsx
   import bundle from 'tegaki/fonts/caveat';
   ```

   Built-in fonts: `caveat`, `italianno`, `tangerine`, `parisienne`.

   Want a different font? Use the [interactive generator](/tegaki/generator/) to create stroke data for any Google Font, then import the downloaded bundle instead:

   ```tsx
   import bundle from './output/my-font/bundle';
   ```

2. **Render the animation**

   Use the `TegakiRenderer` component:

   ```tsx
   import { TegakiRenderer } from 'tegaki';

   function App() {
     return (
       <TegakiRenderer
         font={bundle}
         time={{ mode: 'uncontrolled', speed: 1, loop: true }}
         style={{ fontSize: 48 }}
       >
         Hello World
       </TegakiRenderer>
     );
   }
   ```

</Steps>
